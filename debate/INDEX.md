# Debate Ticket Index

Use this board as the operational queue for all active tickets.

## Update Rules
- Add a row when a ticket is created.
- Update status/owner/priority/last-updated every time the ticket changes.
- Remove from Active Queue when moved to `debate/archive/`.
- Mark `SLA Breached` as `yes` if any status SLA was exceeded.
